# New key bindings

## Login
- Press twice to login

## Main menu
- CTRL + B: Back
- CTRL + Q: Quick access submenu
- CTRL + H: Home
- H: HQ submenu
- V: Event
- B: Battle menu
- CTRL + C: Build menu

## Battle
- Campaign
- Event
- Daily Raids
- Exercises
- Operation Siren

## Campaign
- LARROW: Go to previous chapter
- RARROW: Go to next chapter
- Toggle Normal/Hard Mode
- Shift + 1: 1-1, 3-2, 4-2, 7-1, 9-2
- Shift + 2: 1-2, 2-3, 3-1, 6-3, 8-1, 9-1, 10-1, 11-1, 12-1
- Shift + 3: 1-3, 4-3, 5-2, 6-1, 9-4, 10-3
- Shift + 4: 1-4, 2-2, 3-3, 7-2, 11-3
- Shift + 5: 2-1, 7-4, 8-4, 11-4
- Shift + 6: 2-4, 8-3, 11-2
- Shift + 7: 3-4, 4-4, 5-3, 6-2
- Shift + 8: 4-1, 5-1, 8-2
- Shift + 9: 6-4, 10-2
- Shift + 0: 7-3, 9-3, 10-4
- Shift + A:
- Shift + B:
- Unassigned: 5-4
- G: Go to Fleet Selection
- ENTER: Enter combat
